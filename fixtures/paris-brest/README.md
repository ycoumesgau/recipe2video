# Paris-Brest Demo Fixture

Public-safe fixture for Recipe2Video demo mode. It demonstrates the real product flow without claiming that fixture clips were generated live.

Included data:

- `recipe.json`
- `logical-scenes.json`
- `seedance-segments.json`
- `references.json`
- `prompt-diff-example.json`
- `cost-logs.json`
- `suno-prompt.md`

The matching local demo media lives under `public/demo/paris-brest/`. Real generated media must still be persisted to Supabase Storage and uploaded to Mux for playback.
